Array-Based Queue Implementation in Java
Bu proje, Java dilinde Kuyruk (Queue) veri yapısının bir dizi (array) kullanılarak nasıl oluşturulduğunu gösteren temel bir veri yapıları uygulamasıdır. FIFO (First-In-First-Out / İlk Giren İlk Çıkar) prensibine dayalı çalışma mantığını içerir.
Özellikler
Bu uygulama aşağıdaki temel kuyruk operasyonlarını desteklemektedir:
Enqueue: Kuyruğun arkasına (rear) yeni bir eleman ekleme.
Dequeue: Kuyruğun önünden (front) eleman çıkarma.
Front: Kuyruğun en önündeki elemanı görüntüleme.
Rear: Kuyruğun en sonundaki elemanı görüntüleme.
Dinamik Kontrol: Kuyruğun dolu (isFull) veya boş (isEmpty) olma durumlarını kontrol etme.
